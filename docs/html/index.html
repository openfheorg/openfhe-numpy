Welcome to OpenFHE-NumPy’s documentation!

openfhe-numpy provides a NumPy-like API for homomorphic encryption operations. Built on top of openfhe-python and the OpenFHE C++ library, it enables data scientists and ML practitioners to perform computations on encrypted data using familiar NumPy syntax.

Features

  • NumPy Compatibility - NumPy-like API for encrypted computations
